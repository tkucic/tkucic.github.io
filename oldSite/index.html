Automated ships steam boiler

This project is serving as an implementation of my bachelor degree paper "Automation of ships steam boiler". It covers all the chapters written about in the paper and it even gives the project the HMI and a working simulator.

Second use of this project is to serve as a portfolio of my automation programming skills, product management, hardware and control system design and as such it is changing from year to year.

The control program and the HMI have been implemented with CODESYS 3.5 and serialized with my own IEC61131 serializer for easier versioning with git. The repository is free to view, download and try, but distributing and using it as a base for further product development is not allowed.
