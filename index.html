Automated ships steam boiler
What is a steam boiler:
Essentially, a boiler is a container into which water can be fed and by the application of heat is evaporated continuously into steam. Steam is raised mainly for heating the heavy fuel oil used to fire the ‘diesel engines’ , for space heating , to power cargo handling pumps and as an aid to cleaning storage tanks and surfaces.
This program controls the ignition sequence of the burner in order to create steam. Along with the boiler program a simulator is included to simulate loads and to test the PID regulation.

Showcase video (v2.0): Here

Goal of this program
The idea behind of this project is to provide a support project for my bachelor degree paper "Automation of ships steam boiler". When the project is finished the bachelor degree will be accompanied by the complete PLC program and a simulator.

Second intention of the project is to create a reference project that presents my automation skills and as such this project will never be really finished because there is always something to improve.

The project is version controlled at GitHub and can be downloaded from here.

PLC Programming school
Information about Structured Text programming is scarce to say the least so I have decided to solve this problem and dedicate a online school to teaching ST Code to anyone who wants to learn it.
The school is in progress of being built as you are reading this text. It will focus on the IEC61311-3 standard and on Structured Text programming for PLCs. ST Code course will start as a introduction to what it really is and how to use it. After the introduction we will focuse on real life industry problems an we will solve them using structured text programming, functions, simulators and much more.
